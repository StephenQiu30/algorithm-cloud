# 快速排序 (Quick Sort) 深度解析

快速排序（Quick Sort）由 Tony Hoare 在 1960 年提出。它的基本思想是：通过一趟排序将要排序的数据分割成独立的两部分，其中一部分的所有数据都比另外一部分的所有数据都要小，然后再按此方法对这两部分数据分别进行快速排序，整个排序过程可以递归进行。

## 算法原理
1. **挑选基准值 (Pivot)**: 从数列中挑出一个元素，称为 “基准” (pivot)。
2. **分区 (Partition)**: 重新排序数列，所有比基准值小的元素摆放在基准前面，所有比基准值大的元素摆在基准的后面（相同的数可以到任一边）。在这个分区退出之后，该基准就处于数列的中间位置。
3. **递归排序 (Recursive Sort)**: 递归地（recursive）把小于基准值元素的子数列和大于基准值元素的子数列排序。

## 性能分析
- **时间复杂度 (平均)**: $O(n \log n)$。
- **时间复杂度 (最坏)**: $O(n^2)$ - 当分区极度不均衡时（例如已经有序的数组，基准点选在最右侧）。
- **时间复杂度 (最好)**: $O(n \log n)$。
- **空间复杂度**: $O(\log n)$ - 主要是递归调用的栈空间开销。
- **稳定性**: **不稳定** - 分组交换过程中可能打乱相同元素的相对位置。

## 优缺点
- **优点**: 平均性能极佳，是实际应用中最快的通用排序算法。原地排序。
- **缺点**: 最坏情况下性能退化。不稳定。

## Java 实现 (双指针法实现分区)
```java
public class QuickSort {
    public static void sort(int[] arr) {
        if (arr == null || arr.length < 2) return;
        quickSort(arr, 0, arr.length - 1);
    }

    private static void quickSort(int[] arr, int low, int high) {
        if (low < high) {
            int pivotIndex = partition(arr, low, high);
            quickSort(arr, low, pivotIndex - 1);
            quickSort(arr, pivotIndex + 1, high);
        }
    }

    private static int partition(int[] arr, int low, int high) {
        // 简单起见，取最后一个元素为基准。优化建议：使用三数取中法。
        int pivot = arr[high];
        int i = low - 1; 
        for (int j = low; j < high; j++) {
            if (arr[j] < pivot) {
                i++;
                swap(arr, i, j);
            }
        }
        swap(arr, i + 1, high);
        return i + 1;
    }

    private static void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}
```

## 适用场景
- 大规模数据的通用排序场景。
- 绝大多数编程语言的默认库排序实现（如 C 语言的 `qsort`, Java 的 `Arrays.sort` 对 primitive 类型使用的 `Dual-Pivot Quicksort`）。
- **优化技巧**: 随机选择 pivot，或使用“三数取中”避免最坏情况发生。
