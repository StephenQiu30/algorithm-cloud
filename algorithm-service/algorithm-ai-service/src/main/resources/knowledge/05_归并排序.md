# 归并排序 (Merge Sort) 深度解析

归并排序（Merge Sort）是建立在归并操作上的一种有效的排序算法，该算法是采用分治法（Divide and
Conquer）的一个非常典型的应用。归并排序将待排序序列拆分成子序列，直到每个子序列只有一个元素，然后将子序列有序合并。

## 算法原理

1. **分 (Divide)**: 把长度为 $n$ 的输入序列分成两个长度为 $n/2$ 的子序列。
2. **治 (Conquer)**: 对这两个子序列分别采用归并排序（递归）。
3. **合 (Combine)**: 将两个排序好的子序列合并成一个最终的排序序列。

合并过程（Merge）：比较两个有序子序列的第一个元素，将较小的放入辅助数组，然后指针后移，直到其中一个序列为空，最后将另一个序列剩余的部分全部复制到辅助数组。

## 性能分析

- **时间复杂度 (平均/最坏/最好)**: 始终为 $O(n \log n)$。
- **空间复杂度**: $O(n)$ - 需要一个额外的辅助数组来存放合并后的数据。
- **稳定性**: **稳定** - 合并时相同元素的顺序可以被保持。

## 优缺点

- **优点**: 性能非常稳定，不受初始数据分布影响。是稳定性排序。
- **缺点**: 空间开销大（$O(n)$）。

## Java 实现

```java
public class MergeSort {
    public static void sort(int[] arr) {
        if (arr == null || arr.length < 2) return;
        int[] helper = new int[arr.length]; // 辅助数组
        mergeSort(arr, 0, arr.length - 1, helper);
    }

    private static void mergeSort(int[] arr, int left, int right, int[] helper) {
        if (left < right) {
            int mid = left + (right - left) / 2;
            mergeSort(arr, left, mid, helper);       // 左半部分
            mergeSort(arr, mid + 1, right, helper);   // 右半部分
            merge(arr, left, mid, right, helper);    // 合并
        }
    }

    private static void merge(int[] arr, int left, int mid, int right, int[] helper) {
        int i = left;
        int j = mid + 1;
        int k = 0;

        while (i <= mid && j <= right) {
            if (arr[i] <= arr[j]) {
                helper[k++] = arr[i++];
            } else {
                helper[k++] = arr[j++];
            }
        }

        while (i <= mid) helper[k++] = arr[i++];
        while (j <= right) helper[k++] = arr[j++];

        // 将合并后的结果回填到原数组
        for (int m = 0; m < k; m++) {
            arr[left + m] = helper[m];
        }
    }
}
```

## 适用场景

- 外部排序（处理比内存大的数据时）。
- 要求算法性能稳定且不在乎额外空间开销的场景。
- Java 集合框架中的 `Arrays.sort(T[] a)` 针对对象数组使用的 `TimSort` 就是归并排序的一种变体。
